# Portfolio

este repositório contém o meu primeiro portfólio desenvolvido utilizando HTML, CSS e JavaScript.  
O projeto foi criado durante os cursos da Oracle Next Education em parceria com a Alura.

## Sobre o projeto

A proposta era construir um portfólio pessoal simples, funcional e responsivo, aplicando os conceitos iniciais de desenvolvimento web.  
aqui estão organizados meus primeiros estudos com estruturação de páginas, estilização e um pouco de lógica no front-end

## Acesso ao site

você pode visualizar o portfólio em funcionamento através do link:  
https://portfolio-nine-phi-30.vercel.app

## Tecnologias utilizadas

- HTML  
- CSS  
- JavaScript  

## Objetivo do projeto

O foco deste trabalho foi:
- Criar minha primeira página pessoal do zero
- Praticar fundamentos de layout e responsividade
- Entender a divisão entre estrutura, estilo e comportamento
- Publicar um projeto utilizando hospedagem gratuita

## Estrutura

O repositório contém:
- Arquivo HTML principal  
- Estilos organizados em CSS  
- Scripts JavaScript básicos utilizados para funcionalidades simples

## Observação

por ser meu primeiro portfólio, o projeto representa minha fase inicial de aprendizado e serviu como base para evoluções futuras em front-end.
